# 📱 Play Store App Review Analysis

An end-to-end Data Analytics project that analyzes Google Play Store applications and user reviews to uncover valuable business insights through data cleaning, exploratory data analysis (EDA), and visualization.

---

## 📌 Project Objective

The objective of this project is to analyze Google Play Store apps and user reviews to understand:

- App ratings distribution
- Most reviewed applications
- User sentiment
- Popular categories
- Installation trends
- Business insights for developers

---

## 📂 Dataset

Two datasets were used:

1. Play Store Data
2. User Reviews

Dataset contains information such as

- App Name
- Category
- Rating
- Reviews
- Size
- Installs
- Price
- Genres
- User Sentiment
- Sentiment Polarity

## 🔍 Key Insights

✔ Most applications have ratings between **4.0–4.5**

✔ Facebook is the most reviewed application.

✔ Positive reviews significantly outnumber Neutral and Negative reviews.

✔ Free applications dominate the Play Store.

✔ Communication, Tools and Entertainment categories have the highest number of applications.
